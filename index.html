企业文档管理系统

Enterprise Document Management System

基于 FastAPI 的局域网文档管理平台

JWT 认证 四级权限 批复系统 FastAPI

核心功能

用户分级

四级用户体系：系统管理员、部门主管、普通员工、访客

直属关系

支持上下级关系树，上级可管理下属文档和账户

权限控制

文档三级可见性：私有、部门、公开，细粒度权限控制

批复系统

支持@提及用户，批复状态跟踪（待处理/已解决）

技术架构

后端

  • Python 3.8+
  • FastAPI 框架
  • Uvicorn ASGI
  • JWT 认证
  • SHA256 密码哈希

前端

  • HTML5
  • TailwindCSS
  • JavaScript (ES6+)
  • Font Awesome 图标
  • 响应式设计

存储

  • JSON 用户数据
  • CSV 文档元数据
  • JSONL 批复记录
  • 文件系统存储
  • 无需数据库

快速开始

1 下载项目

git clone <项目地址>
cd lan_doc_system

2 安装依赖

pip install -r requirements.txt

3 启动服务

# Windows
start.bat

# Linux/Mac
./start.sh

# 或手动启动
python main.py

4 访问系统

打开浏览器访问: http://localhost:8000

默认测试账号

系统管理员

admin / admin123

部门主管

manager1 / manager123

普通员工

staff1 / staff123

访客
